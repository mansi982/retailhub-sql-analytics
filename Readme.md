# 🛒 RetailHub: End-to-End SQL Data Analytics Project

**MySQL • SQL • Database Design • Data Analytics**



## 📖 Overview

RetailHub is an end-to-end MySQL Data Analytics project that simulates a
real-world e-commerce business. It demonstrates database design,
normalization, data loading, advanced SQL programming, and business
analytics.

## 🚀 Features

* 22 Normalized Tables (3NF)
* Primary \& Foreign Keys
* CHECK, UNIQUE \& DEFAULT Constraints
* Views
* Stored Procedures
* User Defined Functions
* Triggers
* Window Functions
* CTEs
* Business KPI Queries
* Complete Documentation \& Diagrams

## 🎯 Business Problem

RetailHub helps answer business questions such as: - Which products
generate the highest revenue? 
- Which customers are most valuable? 

\-Which products require restocking?
- Which delivery agents perform best? 
- How effective are coupons? 
- How is customer support performing?
